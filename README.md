# Snake Game in Terminal using Nodejs

![demo](https://github.com/iamajraj/snake-game-nodejs/blob/main/demo.gif?raw=true)

Play snake 🐍 game in your own terminal.

## To play

Run the command

```bash
   npm start
```

### Controls

- `🡡`   **`Up Key`**
- `🡢`   **`Left Key`**
- `🡣`   **`Down Key`**
- `🡠`   **`Right Key`**
